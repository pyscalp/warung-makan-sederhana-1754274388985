# Warung Makan Sederhana

Website profesional untuk Food & Beverage yang dibangun dengan teknologi modern.

## Teknologi
- Vite (Build Tool)
- Tailwind CSS (Styling)
- JavaScript ES6+ (Programming)

## Fitur
- Responsive design
- SEO optimized
- Fast loading
- Modern UI/UX

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
